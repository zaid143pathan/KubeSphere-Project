5. Disabled Swap

We executed:

sudo swapoff -a

and permanently disabled swap.

Why?

Kubernetes scheduler requires accurate memory management.

Swap can break:

pod scheduling
performance
kubelet behavior

So Kubernetes requires:

Swap = OFF

8. Installed containerd

You installed:

containerd

What Is containerd?

Container runtime is responsible for:

✅ running containers
✅ pulling images
✅ managing container lifecycle

Kubernetes DOES NOT directly run containers.

Instead:

Kubernetes
    ↓
containerd
    ↓
Containers

What Is Control Plane?

scheduling

scaling
node management
3. Configured kubectl Access

We configured:

mkdir -p $HOME/.kube

4. Installed Calico CNI

We installed:

Calico

using:

kubectl apply -f calico.yaml
What Is Calico?

Calico is a:

CNI (Container Network Interface)

It provides:

✅ pod networking
✅ inter-node communication
✅ pod-to-pod connectivity
✅ network routing

Why Is CNI Important?

Without CNI:

❌ pods cannot communicate
❌ cluster networking fails
